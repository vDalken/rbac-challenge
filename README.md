## Description

This project implements a Role-Based Access Control (RBAC) system using NestJS, TypeORM and PostgreSQL. The RBAC system manages user roles and permissions to ensure secure access to resources within an application.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
